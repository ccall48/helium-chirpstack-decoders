S2100 Datalogger

fPort: 3

Set the Data Uplink Interval
0x80 0x00 0x00 duty_H duty_L

Example: Set the Nodeâ€™s data interval is 10 minutes. Send the downlink command (HEX) via FPort=3: 80 00 00 00 0A

Command List:

0x80 Fixed field
0x00 Fixed field
0x00 Fixed field
duty_L Data interval low byte, you can set the data interval, unit: minute
duty_H Data interval high byte, you can set the data interval, unit: minute
